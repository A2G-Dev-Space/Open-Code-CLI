# Advanced Usage

## 세션 관리

대화 히스토리는 자동으로 저장됩니다. 이전 세션을 불러오려면:

```
/load
```

세션 목록이 표시되고 원하는 세션을 선택할 수 있습니다.

## 컨텍스트 관리

긴 대화 중 컨텍스트가 80%를 넘으면 자동으로 압축됩니다.

수동 압축:
```
/compact
```

## 로그 모드

디버깅을 위한 상세 로그:

```bash
# 상세 로그 활성화
nexus --verbose

# 디버그 모드
nexus --debug
```

## 도구 사용 통계

```
/usage
```

현재 세션의 토큰 사용량과 도구 호출 통계를 확인할 수 있습니다.
