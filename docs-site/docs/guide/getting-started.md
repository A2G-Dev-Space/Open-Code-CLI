# Getting Started

Nexus Coder를 시작하는 방법을 안내합니다.

## 요구 사항

- Node.js 18 이상
- LiteLLM 또는 OpenAI-Compatible API 서버

## 설치

```bash
npm install -g nexus-coder
```

## 초기 설정

```bash
nexus config init
```

설정 마법사가 실행되어 API 엔드포인트와 모델을 선택할 수 있습니다.
