# Basic Usage

## 기본 명령

```bash
# 대화형 모드 시작
nexus

# 직접 질문하기
nexus "이 프로젝트 구조 설명해줘"
```

## 대화형 모드

<!-- 스크린샷 추가 예정 -->

대화형 모드에서 사용할 수 있는 명령어:

| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 표시 |
| `/clear` | 대화 초기화 |
| `/compact` | 컨텍스트 압축 |
| `/config` | 설정 변경 |
| `Ctrl+C` | 실행 중단 |
