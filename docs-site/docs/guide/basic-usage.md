# Basic Usage

## 기본 명령

```bash
# 대화형 모드 시작
nexus

# 상세 로깅
nexus --verbose

# 디버그 모드
nexus --debug
```

## 대화형 모드

대화형 모드에서 자연어로 요청을 입력합니다:

```
> src 폴더의 구조를 알려줘
> package.json에서 버전을 확인해줘
> login 함수의 버그를 수정해줘
```

## Slash Commands

| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 표시 |
| `/clear` | 대화 초기화 |
| `/compact` | 대화 압축 (컨텍스트 절약) |
| `/load` | 저장된 세션 불러오기 |
| `/model` | LLM 모델 전환 |
| `/tool` | Optional Tools 활성화/비활성화 (Browser 등) |
| `/settings` | 설정 메뉴 |
| `/usage` | 토큰 사용량 통계 |
| `/docs` | 문서 관리 |

## 키보드 단축키

| 키 | 기능 |
|----|------|
| `Tab` | Auto ↔ Supervised 모드 전환 |
| `@` | 파일 브라우저 |
| `/` | 명령어 자동완성 |
| `ESC` | 현재 작업 중단 |
| `Ctrl+C` | 종료 |

## 모델 전환

관리자가 등록한 LLM 모델 중 원하는 모델로 전환할 수 있습니다:

```
/model
```

모델 목록이 표시되고 선택하면 즉시 전환됩니다.

## 작업 모드

### Auto Mode
파일 수정, 명령 실행 등을 자동으로 수행합니다.

### Supervised Mode
파일 수정 전 사용자 승인을 요청합니다. `Tab` 키로 전환 가능합니다.

## 작업 예시

### 코드 수정
```
> login 함수의 버그를 수정해줘
> 이 함수에 에러 처리를 추가해줘
```

### 파일 생성
```
> 새로운 API 엔드포인트를 만들어줘
> UserService 클래스를 생성해줘
```

### 프로젝트 탐색
```
> 이 프로젝트에서 인증 관련 코드가 어디에 있어?
> 데이터베이스 연결 설정을 찾아줘
```

### 코드 분석
```
> 이 함수가 무슨 일을 하는지 설명해줘
> 성능 개선점이 있는지 분석해줘
```
