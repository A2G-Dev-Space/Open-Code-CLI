::: tip Adding Videos
Place your `.mp4` files in `docs/public/videos/` directory:

```
docs/public/videos/
├── codebase-understanding.mp4
├── issue-fixing.mp4
└── documentation.mp4
```

Then reference them in markdown:
```html
<video controls width="100%">
  <source src="/videos/your-video.mp4" type="video/mp4">
</video>
```
:::
