## 활용 예시

### 코드 주석 작성

```
> 이 함수에 JSDoc 주석을 추가해줘
> 이 클래스의 각 메서드에 설명을 달아줘
> 복잡한 로직에 인라인 주석을 추가해줘
```

### API 문서 생성

```
> 이 API 엔드포인트들의 문서를 작성해줘
> Swagger/OpenAPI 스펙을 생성해줘
> 각 API의 요청/응답 예시를 만들어줘
```

### README 작성

```
> 이 프로젝트의 README를 작성해줘
> 설치 방법과 사용법을 문서화해줘
> 환경 변수 설정 가이드를 만들어줘
```

### 기술 문서 작성

```
> 이 모듈의 아키텍처 문서를 작성해줘
> 데이터베이스 스키마 문서를 만들어줘
> 배포 프로세스를 문서화해줘
```

## 팁

- 기존 코드 스타일에 맞게 문서를 생성합니다
- 영어/한국어 지정이 가능합니다: "한국어로 README 작성해줘"
- 마크다운, JSDoc, TypeDoc 등 다양한 형식을 지원합니다
