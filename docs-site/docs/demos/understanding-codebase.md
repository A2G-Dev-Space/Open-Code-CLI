## 활용 예시

### 프로젝트 구조 파악

```
> 이 프로젝트의 전체 구조를 설명해줘
> src 폴더 아래 주요 모듈들이 뭐가 있어?
```

### 아키텍처 이해

```
> 이 프로젝트의 아키텍처를 분석해줘
> 데이터 흐름이 어떻게 되는지 설명해줘
```

### 특정 기능 찾기

```
> 인증 관련 코드가 어디에 있어?
> API 엔드포인트들을 모두 찾아줘
> 데이터베이스 연결 설정은 어디서 해?
```

### 의존성 분석

```
> 이 모듈이 어떤 다른 모듈에 의존하고 있어?
> UserService를 사용하는 파일들을 찾아줘
```

## 팁

- 처음 프로젝트를 열면 전체 구조부터 물어보세요

- `@` 키를 눌러 파일 브라우저로 직접 탐색할 수도 있습니다
